### egg 是一个 MVC 框架

- view 视图 模板 页面的展示
- controller  控制器  负责一些业务逻辑的处理
- model  模型  和数据打交道(查询数据库 请求数据)

### 安装
`yarn global add egg-init`
### 创建项目
`egg-init pro --type=simple`
